# 报表口径与指标说明

## 文档作用
本文件用于补充阶段二的报表知识库，给 AI 提供经营看板和报表分析页面中各项指标、趋势图、占比图、周转图和账龄分析的统一解释口径。

## 权限边界
1. 报表相关只读查询依赖 `reports.view`。
2. 没有该权限时，AI 不应返回真实经营统计。

## 报表总览指标
1. 累计销售额
   - 口径：`sales_orders.total_amount` 累计求和
   - 排除：已取消订单
2. 累计毛利润
   - 口径：订单明细销售额减去对应商品成本
   - 公式：`quantity * unit_price - quantity * cost_price`
3. 已发货订单
   - 口径：订单状态为 `已发货` 或 `已完成`
4. 低库存商品数
   - 口径：所有商品汇总库存低于 `safe_stock` 的商品数
5. 待回款余额
   - 口径：所有未结清应收的 `remainingAmount` 合计
6. 库存货值
   - 口径：库存数量乘商品成本价后汇总

## 销售趋势图
1. 粒度：按月份汇总。
2. 数据范围：当前年度。
3. 销售额：当月订单金额汇总，排除已取消订单。
4. 毛利润：当月订单明细利润汇总。

## 品类占比图
1. 粒度：按商品分类聚合。
2. 指标：分类销售额。
3. 用途：观察销售结构，不直接代表利润结构。

## 库存周转图
1. 粒度：按商品展示前几名。
2. 周转率口径：销量除以平均库存的简化近似值。
3. 当前实现更偏演示口径，不是严格财务周转率模型。

## 账龄分析
1. 分桶：
   - `30天以内`
   - `31-60天`
   - `61-90天`
   - `90天以上`
2. 应收：按未结清应收余额归桶。
3. 应付：按未结清应付余额归桶。
4. 净额：`应收 - 应付`。

## 当前适合进入知识库的问法示例
1. 累计销售额的统计口径是什么？
2. 报表里的毛利润是怎么计算的？
3. 账龄分析怎么分桶？
4. 库存周转率当前是什么口径？
